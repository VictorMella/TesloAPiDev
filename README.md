# TesloDB API

1. Clonar el proyecto
2. __npm install__
3. Clonar el archivo ```__.env.template__``` y renomnrar la copia a ```__.env__```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```
docker-compose up -d
```
